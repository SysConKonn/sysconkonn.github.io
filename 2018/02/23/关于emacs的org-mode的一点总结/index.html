1 总结开始

1.1 标题：

在文件头加上: 

#+TITLE: ...

注意冒号后面有空格

1.2 作者：

在文件头加上: 

#+AUTHOR: ...

冒号后面有空格

1.3 段落：

这个不用说了，只是用'*'号而已，介个星号就代表几级标题。

1.4 行：

如果在编辑器上换行导出之后经常是没有换行的，如果要换行就要在某一行的最后面打上双反斜线'\\'

1.5 一段文字加粗：

在这一段文字首尾各加一个星号'*',记得在前面的星号前加一个空格
实例: 

*这是粗体*

这是粗体

1.6 删除线：

在首尾各加一个加号'+'
 实例: 

+我被删除了+

我被删除了

1.7 下划线：

这很简单，首尾各加一个下划线''
 实例 

_看我下方的线_

看我下方的线

1.8 斜体

首尾加上斜线就行了。
实例 

/我怎么斜了?/

我怎么斜了?

1.9 下标

在两个之间加一个下划线 实例 

H_2 O

H2 O

顺带提一句，如果想把下划线表示下标关掉的话可以在文件最上面加上命令：

#+OPTIONS: _:nil

注意冒号后面有空格

1.10 上标

使用''符号
实例 

E=mc^2

E=mc2

1.11 等宽字

首尾加上等于号
实例 

=HNFMS=

HNFMS

对比非等宽字：
HNFMS <—等宽
HNFMS <—非等宽

1.12 插入代码块

使用指令: 

请无视 #+BEGIN_SRC ...
   ...
请无视 #+END_SRC

实例 

//使用c++语言（实际上并没有什么用）
请无视 #+BEGIN_SRC C++
#include <bits/stdc++.h>
请无视 #+END_SRC

实际上有快捷方式（我才不会告诉你呢）
只要新起一行，输入'<s',然后按'tab'键就可以出来框架 

<s  RET  tab

1.13 打开日历

对，你没听错就是 日历
快捷键: 

C-c C-d

然后会出现类似于下面的东西，只不过是在当前标题的下面一行

DEADLINE: <2018-02-19 一>

1.14 添加待做事项

加Ｎ个星号'*'然后空一格加上'TODO'然后再空一格写待做事项
实例 

为了防止被吞，我先加个井号
#  *** TODO Go for dinner  #

1.14.1 TODO Go for dinner

1.15 待做事项已完毕

在添加了待做事项的那一行按下快捷键：

C-c C-t

效果：'TODO'变成了'DONE'表示你做完了。

1.15.1 TODO Go for dinner

1.15.2 DONE Go for dinner

实际可以更改显示的字，具体怎么做懒得说了。
